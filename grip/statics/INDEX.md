# Grip Statics

[← Grip](../INDEX.md) · [← Home](../../README.md)

Statics covers the physical placement of the hands on the club at address — before any motion begins. These positions set the starting conditions for the entire swing.

## Grip Strength

"Strength" here doesn't mean how hard you squeeze — it refers to how rotated a hand is on the club relative to a baseline position.

| File | Description |
|------|-------------|
| [Weak](weak.md) | Hand rotated toward the target — fewer knuckles visible (left) or palm more on top (right) |
| [Neutral](neutral.md) | Baseline position — 2 knuckles visible (left), lifeline covers thumb (right) |
| [Strong](strong.md) | Hand rotated away from the target — more knuckles visible (left) or palm more under (right) |
