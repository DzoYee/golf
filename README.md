# Golf Knowledge Base

A structured, cross-linked knowledge base covering the fundamentals of the golf swing. Built to be read on GitHub — every concept lives in its own file, linked inline where it's relevant and discoverable through navigation hubs.

## How to Navigate

- **INDEX.md** files at each folder level act as navigation hubs — start there to browse a topic area
- **Cross-links** appear inline where concepts connect (e.g., a grip file links to the dynamics it influences)
- **See Also** footers on each file surface related reading
- **[manifest.json](manifest.json)** tracks the full topic inventory, relationships between concepts, and open questions

## Conventions

- All links use **relative paths** so they resolve on GitHub
- Each file is **self-contained** — you shouldn't need to read a parent file to understand a child
- The manifest tracks **relationships** between topics (e.g., "strong grip tends to close the face at impact")
- Open questions use a `GQ-n` ID scheme and live in the manifest
